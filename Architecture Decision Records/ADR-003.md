# ADR-003: Постановка 

## Контекст
В Avito две колонки категорий: parent_category_name (9 классов) 
и category_name (47 классов). Между ними есть иерархия. 
Нужно решить, как будем решать задачу :) 

## Варианты
1. **Flat на parent (9 классов):** Просто, но теряем 
   детализацию и не показываем работу с длинным хвостом.
2. **Flat на child (47 классов).** Стандартная классификация по 47 классам, игнорируем иерархию. 
3. **Каскад (parent->child).** 10 моделей, ошибка на parent критическая + дорого по коду и времени.
4. **Multi-task (одна модель, две головы - Bert с 2-мя головами).** Совместное 
   обучение на parent и child

## Решение
**Изначально** планировался multi-task BERT. Из-за бюджета времени реализован flat BERT на 47 классов
