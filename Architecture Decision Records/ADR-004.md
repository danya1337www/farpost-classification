# ADR-004: Baseline Ladder

## Контекст
Без baseline невозможно обосновать, оправдан ли BERT по сравнению с классическими подходами.

## Варианты baseline ladder 
1. Только BERT (без baseline) - нельзя обосновать, неизвестно стоит ли своих ресурсов
2. Только классические модели - не использует возможности 
   современных трансформеров (в этом случае BERT)
3. Лестница из 3 уровней

## Решение
**LVL 0:** TF-IDF + LinearSVC. Чистый текстовый baseline 
(только title + description).

**LVL 1:** CatBoost с табличными фичами (price, region, 
city, длина текста и тд. ) (планировалось с text_features, но решено не использовать)

**LVL 2:** Multi-task BERT: 
rubert-tiny2 как стартовая, fine-tuning. 
multilingual-e5-base - при апргейде.

## Обоснование
- Каждый уровень даёт reference point для следующего.
- Прирост между уровнями - сравнение "результатов" 
  в отчёте.
## Out of scope
- Мультимодальность (текст + картинки).
* **UPD** - e5-base
