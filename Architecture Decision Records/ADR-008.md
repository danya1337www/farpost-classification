# ADR-008:  scope 

## Контекст
Тестовое имеет ограничение по времени - 7 дней. 

## В рамках scope
- EDA датасета Avito
- Stratified sample 300k со split 80/10/10
- Ladder из 3 моделей (TF-IDF+LinearSVC, CatBoost, multi-task (решено не делать) BERT)
- Class weights
- F1-macro как основная метрика, two-level evaluation
- Error analysis (минимум)
- Отчёт с обоснованием решений и планом улучшений

## Out of scope (вынесено в "будущие улучшения")
1. **Мультимодальность (текст + картинки + табличные).**
2. **Fusion BERT + tabular (Level 3).** 
3. **Маппинг категорий Avito -> Farpost.** Требует парсинга 
   Farpost и transfer learning поверх обученной модели.
4. **Каскадная архитектура (per-parent classifiers).**
